# Vosi i18n（zh / en / ja）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use tre:subagent-driven-development (recommended) or tre:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Chinese (default), English, and Japanese UI localization across all user-visible surfaces, switchable from Settings → General → Language on save.

**Architecture:** Frontend and Rust each maintain `locales/{zh,en,ja}.json` with shared key naming. `general.locale` persists in `settings.toml`. On save, Rust `apply_locale` rebuilds tray menu/tooltip, emits `locale-changed`, and updates window titles; WebViews re-render via `t()` + `data-i18n`.

**Tech Stack:** Tauri 2, Rust, vanilla TypeScript, Vite multi-page build, `serde_json` (already in workspace via other deps — use existing `serde_json` in Cargo.toml)

**Spec reference:** `docs/specs/2026-06-08-vosi-i18n-design.md`

```rust
pub mod locale;

use locale::Locale;
use std::collections::HashMap;
use std::sync::LazyLock;

static CATALOG: LazyLock<HashMap<Locale, HashMap<String, String>>> = LazyLock::new(|| {
    let mut map = HashMap::new();
    map.insert(Locale::Zh, parse(include_str!("locales/zh.json")));
    map.insert(Locale::En, parse(include_str!("locales/en.json")));
    map.insert(Locale::Ja, parse(include_str!("locales/ja.json")));
    map
});

fn parse(raw: &str) -> HashMap<String, String> {
    serde_json::from_str(raw).expect("parse locale json")
}

pub fn t(locale: Locale, key: &str) -> String {
    t_with_vars(locale, key, &[])
}

pub fn t_with_vars(locale: Locale, key: &str, vars: &[(&str, &str)]) -> String {
    let catalog = CATALOG.get(&locale).or_else(|| CATALOG.get(&Locale::Zh));
    let mut text = catalog
        .and_then(|c| c.get(key))
        .cloned()
        .or_else(|| CATALOG.get(&Locale::Zh).and_then(|c| c.get(key)).cloned())
        .unwrap_or_else(|| {
            eprintln!("missing i18n key: {key}");
            key.to_string()
        });
    for (name, value) in vars {
        text = text.replace(&format!("{{{name}}}"), value);
    }
    text
}
```

In `tray.rs`, add:

```rust
use crate::i18n::{self, locale::Locale};
use std::sync::atomic::{AtomicU8, Ordering};

static TRAY_STATUS: AtomicU8 = AtomicU8::new(0); // 0=Idle, 1=Recording, 2=Warning

pub fn tooltip_for(status: TrayStatus, locale: Locale) -> String {
    let key = match status {
        TrayStatus::Idle => "tray.tooltip.idle",
        TrayStatus::Recording => "tray.tooltip.recording",
        TrayStatus::Warning => "tray.tooltip.warning",
    };
    i18n::t(locale, key)
}

pub fn set_status<R: Runtime>(app: &AppHandle<R>, status: TrayStatus) {
    TRAY_STATUS.store(status as u8, Ordering::Relaxed);
    let locale = app.state::<crate::app::state::AppState>().locale();
    if let Some(tray) = app.tray_by_id("main") {
        let _ = tray.set_tooltip(Some(tooltip_for(status, locale)));
        // icon unchanged
    }
}

pub fn rebuild_tray_menu<R: Runtime>(app: &AppHandle<R>, locale: Locale) -> Result<(), tauri::Error> {
    let show = MenuItem::with_id(app, "show", &i18n::t(locale, "tray.menu.settings"), true, None::<&str>)?;
    let about = MenuItem::with_id(app, "about", &i18n::t(locale, "tray.menu.about"), true, None::<&str>)?;
    let quit = MenuItem::with_id(app, "quit", &i18n::t(locale, "tray.menu.quit"), true, None::<&str>)?;
    let menu = Menu::with_items(app, &[&show, &about, &quit])?;
    if let Some(tray) = app.tray_by_id("main") {
        tray.set_menu(Some(menu))?;
        let status = match TRAY_STATUS.load(Ordering::Relaxed) {
            1 => TrayStatus::Recording,
            2 => TrayStatus::Warning,
            _ => TrayStatus::Idle,
        };
        let _ = tray.set_tooltip(Some(tooltip_for(status, locale)));
    }
    Ok(())
}
```

- [ ] **Step 3: Create `src/i18n/index.ts`**

```typescript
import type { Locale } from "./types";
import { parseLocale } from "./types";
import zh from "../locales/zh.json";
import en from "../locales/en.json";
import ja from "../locales/ja.json";

const catalogs: Record<Locale, Record<string, string>> = { zh, en, ja };

let currentLocale: Locale = "zh";

export function setLocale(locale: Locale | string): void {
  currentLocale = parseLocale(locale);
}

export function getLocale(): Locale {
  return currentLocale;
}

export function t(key: string, vars?: Record<string, string>): string {
  let text =
    catalogs[currentLocale][key] ??
    catalogs.zh[key] ??
    (console.warn(`missing i18n key: ${key}`), key);
  if (vars) {
    for (const [name, value] of Object.entries(vars)) {
      text = text.replaceAll(`{${name}}`, value);
    }
  }
  return text;
}

export function applyLocale(): void {
  document.documentElement.lang =
    currentLocale === "zh" ? "zh-CN" : currentLocale === "ja" ? "ja" : "en";

  document.querySelectorAll<HTMLElement>("[data-i18n]").forEach((el) => {
    const key = el.dataset.i18n!;
    el.textContent = t(key);
  });

  document.querySelectorAll<HTMLOptionElement>("option[data-i18n]").forEach((el) => {
    const key = el.dataset.i18n!;
    el.textContent = t(key);
  });

  const titleKey = document.body.dataset.windowTitle;
  if (titleKey) {
    document.title = t(titleKey);
  }
}
```
