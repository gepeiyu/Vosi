# Vosi 手动测试清单

**应用版本**：`0.1.1`（与 `Vosi_0.1.1_*.dmg` / NSIS 安装包版本号一致）

在发布前于 **macOS** 和 **Windows** 各完成一遍。当前 ASR 为 **SenseVoice INT8**（多语 `auto`），含可选标点后处理。

## 安装与启动

- [ ] 应用安装后首次启动无崩溃
- [ ] 托盘图标可见（若 `show_tray = true`）
- [ ] 右键托盘 →「设置」可打开设置窗口
- [ ] 右键托盘 →「退出」可正常退出

## 权限

- [ ] macOS：系统设置 → 隐私与安全性 → **麦克风** 已授权
- [ ] macOS：系统设置 → 隐私与安全性 → **辅助功能** 已授权（文本注入）
- [ ] Windows：设置 → 隐私 → **麦克风** 已允许桌面应用访问

## 按住说话（Hold-to-Talk）

- [ ] 默认热键（macOS 右 Command / Windows 右 Alt）**按住 ≥ 300 ms** 开始录音，托盘 tooltip 显示「正在录音…」
- [ ] 松开热键后停止录音，tooltip 恢复「就绪」
- [ ] 短按（< 300 ms）不产生输出、不显示浮窗
- [ ] macOS：左 Command + C 等快捷键**不**唤起 Vosi
- [ ] 对着麦克风说「你好世界」，松开后文本注入到当前焦点应用
- [ ] 中英混说一句（如「打开 Chrome 浏览器」），识别含英文词
- [ ] 首次启动：模型从 bundled 复制到 Application Support 期间，托盘显示准备中（非「就绪」）

## 文本注入

- [ ] macOS TextEdit / Windows 记事本：`type` 模式逐字输入正确
- [ ] `paste` 模式通过剪贴板粘贴正确
- [ ] 注入失败时托盘进入警告状态

## 设置持久化

- [ ] 修改触发键、ASR 模式、静音阈值、注入方式、热词路径后保存
- [ ] 重启应用后设置仍生效（`~/.config/vosi/settings.toml`）

## 离线合规

- [ ] 断网状态下可完成一次完整 dictation（模型已本地安装）
- [ ] 日志文件 `~/Library/Application Support/vosi/logs/vosi.log`（macOS）不含识别文本内容

## 浮动胶囊

- [ ] 按住热键：底部居中胶囊出现，音量条随说话跳动
- [ ] 松开热键：胶囊切换「识别中…」脉冲
- [ ] 上屏成功后胶囊隐藏
- [ ] 设置关闭「显示浮动胶囊」后按住无胶囊（管线仍工作）

## 错误通知

- [ ] 注入失败：系统通知「已复制到剪贴板」+ 托盘警告图标
- [ ] 断开麦克风后按住：通知「未检测到麦克风」

## 长句模式

- [ ] 设置 ASR 模式为「长句」，说 20s+ 带停顿语音，上屏文本含换行

## 托盘图标

- [ ] 就绪 / 录音 / 警告 三态图标可区分
- [ ] 就绪态为**白色胶囊 + 红点 + 蓝柱**，透明底，与菜单栏其他图标高度协调
- [ ] App 图标（Dock）与托盘图标视觉一致但比例可不同

## 打包（Release 构建后）

- [ ] macOS arm64 / Intel `.dmg` 可安装并运行（CI 或对应架构机器本地构建）
- [ ] Windows NSIS `.exe` 可安装并运行
- [ ] 安装包内含 bundled 模型（`tauri.conf.json` 使用 `models/bundled/` 递归打包；构建前运行 `prepare-bundle-models.sh`）

## 多语言（zh / en / ja）

- [ ] 首次安装 / 无 locale 字段的旧配置 → 界面中文
- [ ] 设置 → 语言 → English → 保存 → 设置页全英文
- [ ] 托盘菜单、tooltip 同步英文
- [ ] 关于页英文
- [ ] 按住热键 → 胶囊显示「Processing」
- [ ] 权限区块 label / 按钮英文
- [ ] 切换日文重复上述检查
- [ ] 重启应用 → 语言保持
- [ ] 语言下拉始终显示 `中文` / `English` / `日本語`（不随当前 UI 语言变化）
